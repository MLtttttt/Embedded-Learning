
## 固定模板

```c
#include "reg52.h"

sbit LED = P2^0;     // 定义引脚：LED接在P2.0（根据实际电路修改）

// 延时函数
void delay(unsigned int n) {
    unsigned int i, j;
    for (i = 0; i < n; i++)
        for (j = 0; j < 120; j++);   // 120 ≈ 1ms (12MHz),经验所得
}

void main() {
    while (1) {
        LED = 0;      // 低电平点亮LED（共阳极接法）
        delay(500);   // 亮500ms
        LED = 1;      // 高电平熄灭LED
        delay(500);   // 熄500ms
    }
}

```
**注意**：如果开发板是 **共阴极**（高电平点亮），请将 `LED = 0` 和 `LED = 1` 互换。
