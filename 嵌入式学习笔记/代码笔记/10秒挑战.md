# 秒表计时及成绩评估系统

可用于100米体育成绩测试
系统具备计时、显示、成绩评估及串口输出功能。
用户可通过按键控制计时启停与清零，计时暂停时根据成绩区间点亮不同数量的LED指示灯
并通过串口发送时间及等级评定。


### 引脚分配表

| 外设     | 引脚        | 说明         |
| ------ | --------- | ---------- |
| 数码管段码  | P0.0~P0.7 | 共阴极数码管段码输出 |
| 数码管位选  | P2.0~P2.5 | 高电平选中对应位   |
| 启动/暂停键 | P3.2      | 低电平按下      |
| 清零键    | P3.3      | 低电平按下      |
| LED1   | P1.0      | 高电平点亮      |
| LED2   | P1.1      | 高电平点亮      |
| LED3   | P1.2      | 高电平点亮      |


###  系统模块划分
程序主要分为以下模块：
- **主程序**：初始化各模块，循环扫描按键。
- **定时器0模块**：产生1ms中断，用于计时和数码管动态扫描。
- **按键处理模块**：检测启动/暂停键和清零键，带消抖处理。
- **显示模块**：通过定时器中断动态刷新8位数码管，显示格式为“分-秒-毫秒”（实际毫秒显示两位，精度0.01秒）。
- **成绩判断模块**：暂停时根据当前时间计算成绩区间，点亮对应数量LED。
- **串口通信模块**：初始化串口（波特率9600），暂停时发送时间及等级字符串。

### 程序流程图
```
开始

 ├─ 初始化定时器0（1ms中断）

```


### 成绩判断与LED控制
```c
void Judge_LED(void) {
    u32 total_time = (u32)min * 60 * 1000 + (u32)sec * 1000 + msec;
    u32 t = total_time / 10;          // 转换为厘秒（0.01秒）

    if(t >= 950 && t <= 1050) {       // 9.50~10.50s
        LED1 = 1, LED2 = 0, LED3 = 0; // 1颗LED亮
        LED_count = 1;
    }
    else if(t >= 850 && t < 950) {    // 8.50~9.50s
        LED1 = 1, LED2 = 1, LED3 = 0; // 2颗LED亮
        LED_count = 2;
    }
    else if(t >= 750 && t < 850) {    // 7.50~8.50s
        LED1 = 1, LED2 = 1, LED3 = 1; // 3颗LED亮
        LED_count = 3;
    }
    // 其他区间LED保持之前状态，未作处理
}
```

#### 3.3.3 按键扫描（含消抖）
```c
void Key_Scan(void) {
    if(Key_start == 0) {               // 启动键按下
        Delay10ms();                    // 消抖
        if(Key_start == 0) {
            flag = !flag;                // 切换运行/暂停
            if(flag == 0) {               // 暂停时
                Judge_LED();               // 成绩判断
                Uart_result();              // 发送时间
                Send_grade(LED_count);      // 发送等级
                Uart_Send('\r');
                 Uart_Send('\n');
            }
            while(Key_start == 0);        // 等待释放
        }
    }
    else if(Key_clear == 0) {          // 清零键按下
        Delay10ms();
        if(Key_clear == 0) {
            flag = 0;
            min = sec = msec = 0;
            LED1 = LED2 = LED3 = 0;      // 全灭
            while(Key_clear == 0);
        }
    }
}
```

#### 3.3.4 串口发送时间与等级
```c
// 发送时间（格式 MM:SS.dd）
void Uart_result(void) {
    Uart_Send(buf[0] + '0');
    Uart_Send(buf[1] + '0');
    Uart_Send(':');
    Uart_Send(buf[3] + '0');
    Uart_Send(buf[4] + '0');
    Uart_Send('.');
    Uart_Send(buf[6] + '0');
    Uart_Send(buf[7] + '0');
    Uart_Send('\r');
    Uart_Send('\n');
}

// 发送等级字符串
void Send_grade(unsigned char count) {
    if (count == 3) Uart_assess("Excellent!");
    else if (count == 2) Uart_assess("Good!");
    else if (count == 1) Uart_assess("Pass");
    else Uart_assess("Unqualified");
}
```

## 系统使用测试

| 测试项目   | 操作步骤                                   | 预期结果                        |
| ------ | -------------------------------------- | --------------------------- |
| 启动/暂停  | 按下启动键，等待数秒后再次按下                        | 数码管开始计时，暂停时显示当前时间，LED根据成绩点亮 |

| 串口输出   | 暂停时观察串口助手                              | 收到时间及对应等级字符串                |
