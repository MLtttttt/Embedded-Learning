
| 数据类型        | 关键字             | 存储空间 | 取值范围                | 说明                    |
| ----------- | --------------- | ---- | ------------------- | --------------------- |
| **位变量**     | `bit`           | 1 位  | 0 或 1               | 仅用于标志位，高效，不能用于数组/指针   |
| **无符号字符**   | `unsigned char` | 1 字节 | 0 ~ 255             | 最常用，可存小整数、ASCII、数组    |
| **有符号字符**   | `signed char`   | 1 字节 | -128 ~ +127         | 少用，注意负数运算             |
| **无符号整型**   | `unsigned int`  | 2 字节 | 0 ~ 65535           | 用于较大计数、循环、地址          |
| **有符号整型**   | `signed int`    | 2 字节 | -32768 ~ +32767     | 需要负数时使用               |
| **无符号长整型**  | `unsigned long` | 4 字节 | 0 ~ 4.29e9          | 大范围计数、时间累计            |
| **有符号长整型**  | `signed long`   | 4 字节 | -2.147e9 ~ +2.147e9 | 大范围带符号数               |
| **浮点型**     | `float`         | 4 字节 | ±1.18e-38 ~ ±3.4e38 | 小数运算，速度较慢             |
| **双精度浮点**   | `double`        | 4 字节 | 同 float             | C51 中 double 等同 float |
| **特殊功能寄存器** | `sfr`           | 1 字节 | 0x80 ~ 0xFF         | 定义硬件寄存器（如 P0=0x80）    |
| **特殊功能位**   | `sbit`          | 1 位  | 可位寻址的 SFR 位         | 定义引脚（如 P1^0）          |
| **存储类型修饰**  | `code`          | -    | -                   | 将数据放入程序存储器（ROM）       |
| **存储类型修饰**  | `data`/`idata`  | -    | -                   | 指定变量在内部 RAM           |

#### 选择建议
- 标志位：`bit`
- 小整数、字符、数组：`unsigned char`
- 较大计数（如 >255）：`unsigned int`
- 需要负数：`signed int`
- 大范围计数：`unsigned long`
- 小数运算：`float`
- 硬件操作：`sfr` / `sbit`
- 常量表：`code` + 类型（如 `code unsigned char`）
